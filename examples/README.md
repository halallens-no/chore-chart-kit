# Examples

How to use `chore-chart-kit` in real projects.

- `nextjs/` — a minimal Next.js app that renders a template
  server-side
- `html-only/` — a single static HTML file (no build tools)
  that imports the kit via CDN

Both examples link to a sample template and render it as SVG.
